Welcome! Are you completely new to programming and does your name start with E? Then this tutorial is made for you - literally!

The aim is to take you from a complete beginner to a Python pro, in a short amount of time. To prove your Python mastery at the end, you will be tasked with creating a Snake game from scratch. Sounds like fun! Good luck 😎

Tutorial structure

On the left hand side, you will find a list of tutorial chapters. Each chapter introduces new concepts and provides suitable problems for practising. The problems are designed to be used in conjunction with real life tutoring - there is not enough information solely in this course to learn Python from scratch.

Some of the problems have a provided solution which can be viewed. To maximize your learning, you should not view these solutions before trying your best to solve the problem at hand. Even if you cannot figure it out right away, think for a while or ask your tutor before you move on to viewing the actual solution.
