Enter the secret password

In this exercise, you will create a program which repeatedly asks a user for password. You will successively add more functionality to the program.

Part 1: Make a program which asks a user for a secret password. If the password is correct, tell them a supersecret phrase of your choosing.

Part 2: If the user types quit or exit, the program should exit without saying the supersecret phrase.

Part 3: The user should only have 3 attempts to type in the password. If they exceed this amount of attempts, the program should exit.

Part 1 Example Solution 
  password = input("Please enter the secret password: ")
  while password != "supersecretpassword":
      print("Incorrect password! Try again")
      password = input("Please enter the secret password: ")
  print("You entered the correct password, good job! :D")
Part 2 Example Solution 
  password = input("Please enter the secret password: ")
  while password != "supersecretpassword" and password != "quit" and password != "exit":
      print("Incorrect password! Try again")
      password = input("Please enter the secret password: ")

  if password == "supersecretpassword":
      print("You entered the correct password, good job! :D")
  else:
      print("Bye bye, sucker")
Part 3 Example Solution 
  password = input("Please enter the secret password: ")
  tries_left = 2
  while password != "supersecretpassword" and password != "quit" and password != "exit" and tries_left > 0:
      print("Incorrect password! Try again")
      password = input("Please enter the secret password: ")
      tries_left = tries_left - 1

  if password == "supersecretpassword":
      print("You entered the correct password, good job! :D")
  elif tries_left = 0:
      print("Too many failed attempts!")
  else:
      print("Bye bye, sucker")
